# Workflow

## Local dev
- Install: npm install
- Run: npm run dev
- Test: npm test
- Build: npm run build

## Quality gates (current)
- Keep TypeScript build passing.
- Keep unit tests green.

## Context maintenance
- Update product.md when features ship or scope changes.
- Update tech-stack.md when dependencies or infrastructure change.
- Update tracks.md when work starts, pauses, or completes.

## Release notes
- Note visible UI changes and data model changes in track updates.
