# Board Game Setups

Board Game Setups is a small personal project that allows you to select a game and get quick setup steps based on different criteria you choose.

## Local setup

1) Install dependencies:

```bash
npm install
```

2) Create a `.env.local` file in the project root with your Supabase credentials:

```bash
VITE_SUPABASE_URL=your_supabase_project_url
VITE_SUPABASE_ANON_KEY=your_supabase_anon_key
```

3) Run the app:

```bash
npm run dev
```
